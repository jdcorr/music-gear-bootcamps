# music-gear-bootcamps

A growing collection of “bootcamp” guides and practice plans for music hardware.

## Structure

- `bootcamps/dirtywave/` — Dirtywave hardware and software synth/tracker guides
- `bootcamps/teenage-engineering/` — Teenage Engineering hardware guides

## How to use

Each bootcamp is written as a day-by-day practice plan with clear deliverables.

1. Pick a bootcamp folder that matches your gear.
2. Work through the days in order (or jump to the part you need most).
3. Treat the deliverable as the checklist for that day.

## Contributions

If you want to add a new bootcamp:

- Create a new folder under the appropriate vendor namespace.
- Include a `README.md` inside the folder (or name the main doc clearly).
- Keep steps actionable and focused on practice outcomes.
