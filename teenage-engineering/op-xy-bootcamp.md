# OP–XY Bootcamp (14 days) 🎹🔁

Goal: become fast-fluent on the Teenage Engineering OP–XY by building a few coherent patterns (drums → bass → chords) and learning how to shape grooves with step components.

END-OF-BOOTCAMP MASTERY ✅
- You can navigate OP–XY modes/views without getting lost
- You can make a 4/4 drum groove with kick/snare/hi-hat and at least one ratchet variation
- You can record a bassline that stays within (and locks to) your drum groove
- You can place chords rhythmically on a chord/texture track
- You can confidently use pitchbend and understand basic I/O routing (speaker/headphones vs external)

DAILY STRUCTURE ⏱️
- 10 min: navigation + shortcut warmup
- 40–60 min: build today’s deliverable
- 10 min: “what changed & why” notes (tiny)

SHORTCUTS / WORKFLOW MUSCLES 🧠
- Create a new project: press and hold M1
- Track roles (default mapping in new projects):
  - Track 1–2: drums
  - Track 3: bass
  - Track 4: pluck
  - Track 5: lead
  - Track 6: soft pluck
  - Track 7: strings
  - Track 8: pad

DAY 1: START A PROJECT + DRUM TRACK BASICS 🥁
Task:
- Create a new project (hold M1)
- Select Track 1
- Pick the kick key sound and record a 4-on-the-floor using the sequencer buttons
- Press play/stop until you can do it quickly
Deliverable:
- A clean kick pattern you can re-record in under ~1 minute

DAY 2: SNARE + HI-HAT BACKBEAT ✨
Task:
- Select snare key (guide shows using the ‘g’ key adjacent to kick)
- Add snare to the 5th and 13th steps (backbeat), using the record button “view only” trick
- Select hi-hat key (guide shows ‘c#’)
- Place hi-hat on every other step
Deliverable:
- Kick + snare + hi-hat loop that feels like a real groove

DAY 3: STEP COMPONENT RATCHETS (MULTIPLY / DIVIDE) 🔥
Task:
- Select a hi-hat step, then apply a step component:
  - SHIFT + select the step
  - choose “multiply” (natural key) and then change it to “divide into 3 trigs” (accidental key ‘3’ / a#)
- Listen and adjust until the ratchet is musically placed
Deliverable:
- One drum bar where the hi-hat has a ratchet/ratcheted subdivision variation

DAY 4: EXTEND + STRUCTURE (BARS) 🧱
Task:
- Extend your drum pattern to additional bars using BAR + (+)
- Make Phrase A (simple) and Phrase B (variation)
Deliverable:
- A 2–4 bar drum structure with repetition + variation

DAY 5: RECORD A BASSLINE 🎸
Task:
- Select Track 3 (bass)
- Play a bassline over your drum groove (practice)
- Record it (press record to arm; press play/perform notes; stop when done)
- If it’s too long, extend pattern using hold BAR + (+)
Deliverable:
- A bassline that doesn’t “fight” the kick rhythm

DAY 6: AUTOMATION / MOVEMENT FEEL 🎛️
Task:
- While playing back your bassline, hold record and turn one encoder
- If movement is stepped, smooth it using BAR + rotate the white encoder
Deliverable:
- One bassline with expressive encoder movement

DAY 7: CHORDS (RHYTHMIC PLACEMENT) 🎼
Task:
- Select Track 7 (strings/chords)
- Increase track scale so step length matches your chord rhythm (guide example: hold BAR + accidental ‘4’ / c#)
- Play and record chord rhythm while keeping it locked to bars
Deliverable:
- A chord progression that hits exactly when you expect

DAY 8: ADD ONE FX™ MOMENT 😈
Task:
- Add a single punch-in style FX moment in the intro or before a drop (use the OP–XY guide’s FX section for your exact implementation)
Deliverable:
- A noticeable transformation (not just louder)

DAY 9: BUILD A “MINI TRACK” (8–16 bars) 🧩
Task:
- Use at least 3 tracks intentionally (drums, bass, chords/texture)
- Create a simple arrangement: intro → build → loop outro
Deliverable:
- A complete mini track

DAY 10: ADD TEXTURE LAYER 🫧
Task:
- Add a pluck/lead/pad layer (choose one) and make it support the chord rhythm
Deliverable:
- One extra melodic/texture line

DAY 11: TIGHTEN GROOVE 🎚️
Task:
- Improve timing/ratchet placement
- Reduce notes where they don’t help
Deliverable:
- A “cleaner” version of the same track

DAY 12: PRACTICE ROUTING BASICS 🎧
Task:
- Compare internal speaker vs headphones/line out
- Confirm you understand where pitchbend sits and how it behaves
Deliverable:
- You can perform with pitchbend without thinking

DAY 13: POLISH + RECORD 🎙️
Task:
- Record your final performance (if using an external recorder: route to line-in)
Deliverable:
- Final audio you like enough to share

DAY 14: QUALITY PASS ✅
Checklist:
- Does playback sound stable?
- Are patterns readable at a glance?
- Do drums/bass/chords each have a job?
Deliverable:
- A final “Bootcamp” project

---

Safety note (from the OP–XY guide) ⚠️
- Charge using only 5V USB power.
- If connecting/disconnecting wires during use: connect/disconnect with the OP–XY first.
- Never connect the OP–XY input/output 3.5mm plugs to a microphone input with active phantom power.
