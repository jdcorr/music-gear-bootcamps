# TP–7 Bootcamp (14 days) 🎙️📼

Goal: become fast-fluent with the Teenage Engineering TP–7 field recorder by mastering input/source selection, recording workflow, gain/level discipline, editing, and smooth playback/export.

END-OF-BOOTCAMP MASTERY ✅
- You can set up a recording session quickly (source → input config → gain → record)
- You can capture clean audio without distortion
- You can edit recordings (trim/crop-style workflow as supported by the TP–7 UI)
- You can manage files (browse/delete/transfer) without losing work
- You understand playback speed control (and when it’s useful)

DAILY STRUCTURE ⏱️
- 10 min: warmup checks (settings + levels)
- 40–60 min: today’s deliverable practice
- 10 min: “what changed & why” notes

SAFETY + ENVIRONMENT NOTES ⚠️
- Place the TP–7 level and stable.
- Operate in 0–50°C and relative humidity <95%.
- Avoid humid environments (baths/washrooms).
- Never connect the TP–7 3.5mm I/O to a microphone input with active phantom power (can damage the TP–7).
- Charge using 5V USB power.

DAY 1: HARDWARE + BASIC SESSION SETUP 🧭
Task:
- Find the TP–7 controls you’ll use every time: power/volume/speaker, record menu, settings menu, and the source/input selection path.
- Make one short test recording (10–20 seconds) from the built-in microphone or your chosen input.
Deliverable:
- One test file you can play back immediately.

DAY 2: INPUT/SOURCE DISCIPLINE 🎚️
Task:
- Repeat Day 1 but systematically change ONE variable (input/source).
- Keep your recording target the same.
Deliverable:
- Two files (“same performance, different input”), plus 2–3 bullets on which you prefer and why.

DAY 3: GAIN / LEVEL MANAGEMENT 📏
Task:
- Record something with wide dynamics (voice + quiet parts).
- Adjust gain so you avoid distortion while keeping noise low.
Deliverable:
- One conservative file + one pushed closer to the edge.

DAY 4: RECORDING WORKFLOW SPEED ⏩
Task:
- Time yourself to go from “standby” to “recording is happening.”
- Then reduce friction: memorize the fastest path to the record action.
Deliverable:
- A repeatable, fast setup routine.

DAY 5: EDITING A RECORDING ✂️
Task:
- Take one recording and remove mistakes / silence using the TP–7 editing capabilities described in the manual.
Deliverable:
- One edited file that starts/ends exactly how you want.

DAY 6: MEMO RECORDING PRACTICE 📝
Task:
- Use memo recording for quick captures.
- Practice naming / organizing if your workflow uses it.
Deliverable:
- A small “field log” of 3 memos.

DAY 7: PLAYBACK + SPEED CONTROL 🕰️
Task:
- Playback the same file at normal and altered speed (when available).
- Note how pitch/tempo feel and when you’d use it.
Deliverable:
- “When I should use speed change” written in 3 bullets.

DAY 8: LINE-IN OR USB INPUT SESSION (IF APPLICABLE) 🎧
Task:
- If you record external gear, capture line-level output using your chosen routing.
Deliverable:
- One recording from external audio you trust.

DAY 9: FILE MANAGEMENT 📁
Task:
- Browse your recordings, delete one bad take, and keep one good one.
Deliverable:
- A clean library state with only useful takes.

DAY 10: TRANSFER WORKFLOW 🔄
Task:
- Transfer one recording to your computer/phone.
- Verify it plays correctly after transfer.
Deliverable:
- One successfully transferred file.

DAY 11: “GET THE TAKE” REHEARSAL 🎬
Task:
- Record a short planned segment (30–60 seconds).
- Do two takes: one conservative, one creative.
Deliverable:
- Two versions with different intention.

DAY 12: NOISE/ROOM CONTROL 🏠
Task:
- Re-record the same source in a different position (closer/farther or different angle).
Deliverable:
- A comparison note on what improved clarity/noise.

DAY 13: FINAL EDIT + EXPORT/PREP FOR SHARE ✨
Task:
- Pick the best recording, edit it, and make it transfer-ready.
Deliverable:
- One “final” file.

DAY 14: QUALITY PASS ✅
Checklist:
- Distortion? (too hot)
- Noise floor? (too low gain / wrong distance)
- Start/end? (edited)
- Transfer integrity? (file plays after transfer)
Deliverable:
- A mini portfolio of 3 recordings: clean, creative, and one optimized edit.
